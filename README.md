# Lost & Found

A simple **Lost & Found tracker** built with **HTML, CSS, and JavaScript**.  
This app helps you **log, track, and find items** in your home using photos, location tags, and a full history of movements.  

---

## 🌟 Features

- **Add / Move Items** – Log any item with a name, location, and photo.  
- **Item History** – Every movement is tracked with timestamps.  
- **Search Items** – Quickly find an item by name.  
- **Lost vs Found Status** – Recent items (logged within 24 hours) are highlighted in **green**; older items are **red**.  
- **Glass & Gradient UI** – Beautiful gradient background with a modern glass-like centered container.  
- **Fully Client-Side** – Uses `localStorage()`, no backend needed. Works offline in your browser.  

---

## 📸 How It Works

1. Enter **item name**, **location**, and **upload a photo**.  
2. Click **Add / Move Item** to log the item.  
3. Search any item using the search bar.  
4. View the **full history** for each item, including **all previous locations and photos**.  

---

## 🛠 Technologies Used

- **HTML5**  
- **CSS3** (Gradient, Glass Effect)  
- **JavaScript** (localStorage for storage, FileReader for images)
